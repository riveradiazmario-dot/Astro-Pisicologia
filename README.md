# Astro-Pisicologia

AstroAnima — aplicación de Astrología Psicológica Arquetípica.

Proyecto de escritorio web (Vite + React + TypeScript). Contiene motor astronómico, motor interpretativo y exportación a PDF.

Instrucciones rápidas:

1. Instalar dependencias: `npm install`
2. Ejecutar en desarrollo: `npm run dev`

Privacidad: todos los datos se almacenan localmente en IndexedDB.
